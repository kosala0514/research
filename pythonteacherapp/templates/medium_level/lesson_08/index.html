Sets

One of the 4 built-in data type that can use to store multiple items in a single variable.

This is unordered, unchangeable*, and unindexed

Sets cannot have two items with the same value.The values True and 1 are considered the same value in sets, and are treated as duplicates.

To get length of the set

set data types - can be any type


Access set item

We can loop through the set items using a for loop, or ask if a specified value is present in a set, by using the in keyword.

Add set items

We can use add() method

We can add another set using update()

remove set item

To remove an item in a set, use the remove(), or the discard() method. If the item to remove does not exist, remove() will raise an error. But discard() not.

loop through a set

We can use for loop

Join lists

We can use the union() method that returns a new set containing all items from both sets, or the update() method that inserts all the items from one set into another.

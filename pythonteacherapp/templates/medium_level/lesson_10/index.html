if

Indentation is very important

Scenario :

Let's assume this. Today is the session test day. The teacher gives you papers and you answered. After correcting the answer, the teacher enters marks into the system. The teacher wants to know whether you pass or fail the test. (If the mark is greater than or equal to 65, you will pass the test.)

elif

if the previous conditions were not true, then try this condition

Scenario :

A teacher wants to categorize his students according to their marks.

else

Catches anything which isn't caught by the preceding conditions

Scenario :

A student wants to compare two numbers and wants to know what is the greater value.

And

The and keyword is a logical operator, and is used to combine conditional statements.

Or

Test if a is greater than b, OR if a is greater than c.

Not

The not keyword is a logical operator, and is used to reverse the result of the conditional statement.

Nested if

we can have if statements inside if statements, this is called nested if statements.

Pass

if statements cannot be empty, but if you for some reason have an if statement with no content, put in the pass statement to avoid getting an error.
